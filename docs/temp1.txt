then it will branch to E724 instead. The value determine if a button is pressed

E702 by command BL (branch and link similar to x86's CALL). However the subfunction
at 00h:0E752h is too long for me to keep track of.

set a write breakpoint on PC (or SimU8engine.dll+183C which is the command that
read from PC), and each time you use Execute function is equivalent to "Step into"
on the calculator program.
